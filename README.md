# 🪵 Kalkulator Kayu Pro

Sistem Perhitungan Nilai Penjualan Kayu Profesional.

## Fitur
- Input data acuan kubikasi (lingkar, golongan, kubikasi)
- Import & export data acuan via Excel
- Perhitungan otomatis kubikasi & nilai per pohon
- Kalkulator potongan persentase
- Export laporan PDF & Excel
- Riwayat transaksi tersimpan di browser

## Deploy
Aplikasi ini di-host di [Vercel](https://vercel.com) sebagai static site.

## Cara Update
1. Edit file `index.html`
2. Commit & push ke GitHub
3. Vercel otomatis re-deploy
